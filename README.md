# Mumet.in - Jasa Pembuatan Produk Digital

Website ini adalah landing page dan portal layanan untuk Mumet.in, yang menyediakan jasa pembuatan produk digital seperti Website, Aplikasi Mobile, Sistem IoT, Machine Learning, dan UI/UX. Pengunjung bisa melihat layanan, lalu mendaftar/masuk untuk mengirim detail kebutuhan proyek melalui WhatsApp.

## Live Demo

Deploy: https://mumetin.vercel.app/

## Fitur Utama

- Landing page layanan Mumet.in
- Autentikasi pengguna (Email/Password & Google)
- Form pengajuan proyek dan pertanyaan yang terhubung ke WhatsApp
- UI modern dengan animasi dan desain interaktif

## Tech Stack

- Next.js (App Router)
- Firebase Auth
- Tailwind CSS

## Menjalankan Lokal

```bash
npm install
npm run dev
```

Buka http://localhost:3000 di browser.

## Lisensi

Proyek ini milik Mumet.in. Semua hak cipta dilindungi.
